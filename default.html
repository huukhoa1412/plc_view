Xuất khối từ TIA Portal sang PDF

Trong TIA Portal, nhấp chuột phải vào khối đã chọn trong cây dự án và chọn In…. Trong cửa sổ in, đặt máy in là Microsoft Print to PDF. Sau khi chọn cài đặt tệp, nhấp In, rồi lưu tệp PDF. Tên tệp nên trùng với tên khối (ví dụ: Main_OB1.pdf).

Thêm khối vào cây dự án

Trong menu có 3 nút cơ bản. Nhấn Chế độ chỉnh sửa để mở rộng menu, sau đó chọn Thêm khối, điền Tên khối, cung cấp đường dẫn tới PDF (ví dụ: pdf/main.pdf), và xác nhận. Tùy chọn, bạn có thể chọn Tạo F-Block (biểu tượng màu vàng) cho các khối an toàn.

Exporting blocks from TIA Portal to PDF

In TIA Portal, right-click on the selected block in the project tree and choose Print…. In the print window set the printer to Microsoft Print to PDF. After selecting the file settings, click Print, and then save the PDF file. The file name should match the block name (e.g. Main_OB1.pdf).

Adding a block to the project tree

In the menu we have 3 basic buttons. Click Edit Mode to expand the menu, then select Add Block, fill in the Block Name, provide the path to the PDF (e.g. pdf/main.pdf), and confirm. Optionally, you can check Create F-Block (yellow icon) for safety blocks.
